Ark-Models
==========
Arknights Spine Models (Excerpt)  
明日方舟Spine动画小人模型(节选)  

This project only supports Chinese docs. If you are an English user, feel free to contact us.

## 介绍 <sub>Intro</sub>
> 当前更新至：危机合约#11 **赝波行动** 新增的干员和皮肤。  
> 本仓库为不定期更新，欢迎提交PR来完善仓库。  

此仓库收集了《明日方舟》中的部分Spine模型文件，包括：
1. 干员默认基建小人
2. 干员皮肤基建小人
