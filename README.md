Ark-Models
==========
Arknights Spine Models (Excerpt)  
明日方舟Spine动画小人模型(节选)  

**中文** | [English](README-eng.md)

## 介绍
当前更新至：
- 生息演算 **沙中之火** 新增的干员和皮肤

此仓库收集了中国服《明日方舟》中的部分Spine模型文件，包括：
1. 干员默认基建小人
2. 干员皮肤基建小人

> 本仓库为不定期更新，欢迎提交PR来完善仓库。本仓库使用的提取模型文件的工具是[ArkUnpacker](https://github.com/isHarryh/Ark-Unpacker)。
